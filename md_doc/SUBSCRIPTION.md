# 구독 만료 정책

## 무료 체험 기간

| 구분 | 무료 체험 시작 | 만료일 | 알람 시작 | 기능 잠금 |
|------|--------------|--------|----------|----------|
| 5월 15일 이전 가입 | 5월 15일 (고정) | 6월 14일 | 6월 4일 | 6월 15일 |
| 5월 15일 이후 가입 | 가입일 | 가입일 + 30일 | 만료 10일 전 | 만료 다음날 |

- 알람: 만료 10일 이하 남았을 때 배너 표시 (`TRIAL_BANNER_DAYS_THRESHOLD = 10`)
- 기능 잠금: `subscriptionEnd` 이후 `calculatedStatus === "EXPIRED"`

## 만료 후 기능 접근 정책

### 무료로 계속 사용 가능한 탭
| 탭 | 기능 |
|----|------|
| 회원 | 회원 목록 조회, 등록, 수정, 탈퇴 처리 |
| 운동 일정 | 일정 생성, 참석 링크 공유, 참석자 관리 |
| 가입 신청 | 가입 링크 공유, 신청 승인/거절 |
| 회비 | 월회비 관리, 수시회비 관리 (장부 설정 제외) |
| 탈퇴 회원 | 탈퇴 회원 목록, 복구, 영구 삭제 |

### 구독 필요 기능 (만료 시 잠금)
| 기능 | 위치 |
|------|------|
| 자동 대진표 / 코트 배정 | 참석 현황 탭 전체 |
| 통계 | 통계 탭 전체 |
| 장부 관리 설정 | 회비 탭 > 장부 관리 섹션 내 버튼 (🔒 표시, 클릭 불가) |

## 구독 상태 종류

| 상태 | 설명 |
|------|------|
| `TRIAL` | 무료 체험 중 (만료일 이전) |
| `ACTIVE` | 유료 구독 활성 |
| `EXPIRED` | 만료됨 (무료 탭만 접근 가능) |
| `BLOCKED` | 운영자에 의해 차단 (무료 탭만 접근 가능) |
| `EXEMPT` | 무제한 무료 (운영자 지정) |
