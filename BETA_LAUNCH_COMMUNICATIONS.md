# 베타 배포용 문구

## 1. 단톡방 공지용 짧은 안내문

안녕하세요. 클럽 운영을 더 편하게 해보려고 회원 가입 신청, 운동 참석 신청, 회비 관리가 가능한 베타 서비스를 열었습니다.  
아래 링크로 들어가서 가입 신청이나 참석 여부를 남겨주시면 운영진이 더 빠르게 확인할 수 있습니다.

- 가입 신청 링크: `{가입링크}`
- 오늘 운동 참석 링크: `{일정링크}`

아직 베타 버전이라 사용 중 불편한 점이 있으면 편하게 알려주세요.

## 2. 클럽 운영진 공유용 안내문

이번 베타 서비스는 `가입 신청`, `운동 참석/불참`, `회비 관리`, `출석 확인`을 한 화면에서 처리할 수 있도록 만든 운영용 프로그램입니다.  
회원들은 카카오톡 링크를 눌러 직접 신청하고, 운영진은 승인과 예외 처리만 하면 되는 흐름을 목표로 했습니다.

현재 베타에서 우선 확인하고 싶은 부분은 아래입니다.

- 가입 신청 링크가 회원 입장에서 이해하기 쉬운지
- 운동 일정 링크로 참석/불참을 남기는 흐름이 자연스러운지
- 총무 입장에서 명단 정리 시간이 실제로 줄었는지
- 회비와 수시회비 관리가 실사용에 맞는지

베타 기간 동안 받은 피드백을 바탕으로 정식 버전에서 더 다듬을 예정입니다.

## 3. 베타 참여자 피드백 요청 문구

사용해보신 뒤 아래 3가지만 알려주셔도 큰 도움이 됩니다.

1. 가장 편했던 기능은 무엇이었나요?
2. 아직 카카오톡이나 엑셀로 따로 처리해야 했던 일은 무엇이었나요?
3. 계속 쓰고 싶을 정도로 편했는지, 아니라면 어떤 점이 아쉬웠나요?

## 4. 운영진 테스트 체크 문구

아래 순서로 한 번만 테스트 부탁드립니다.

1. 가입 신청 링크 열기
2. 신규 회원 신청 넣기
3. 관리자에서 승인하기
4. 운동 일정 링크 열기
5. 참석 또는 불참 남기기
6. 관리자에서 참석자와 대기자 확인하기
7. 회비 또는 수시회비 체크하기

테스트 중 막히는 지점이 있으면 화면 캡처와 함께 알려주시면 빠르게 반영하겠습니다.

# 사용자 첫 소개 메시지

## 1. 처음 보내는 정중한 버전

안녕하세요. 배드민턴 클럽 운영을 조금 더 편하게 해보려고 베타 프로그램을 하나 만들었습니다.  
지금은 회원 가입 신청, 운동 참석/불참, 회비 체크 같은 기본 운영 업무를 더 쉽게 처리하는 데 초점을 맞추고 있습니다.

아직 초기 버전이라 완벽하진 않지만, 실제 클럽 운영에 맞는 방향으로 계속 다듬고 싶어서 먼저 몇 분께만 조심스럽게 공유드립니다.  
잠깐 써보시고 불편한 점이나 꼭 필요하다고 느끼는 기능을 편하게 말씀해주시면 정말 큰 도움이 됩니다.

필요하시면 테스트 링크와 함께 간단한 사용 순서도 바로 안내드리겠습니다.

## 2. 카톡으로 보내기 쉬운 짧은 버전

안녕하세요. 클럽 운영용 베타 프로그램을 만들었어요.  
가입 신청, 운동 참석/불참, 회비 체크를 링크 기반으로 해볼 수 있는 버전입니다.  
한 번 써보시고 불편한 점이나 있으면 편하게 말씀해주세요. 정말 큰 도움이 됩니다.

## 3. 친한 운영진에게 보내는 편한 버전

이거 요즘 만들고 있는 클럽 운영 프로그램인데, 드디어 한 번 써볼 수 있는 정도까지 왔어.  
가입 신청이랑 운동 참석 링크를 카톡으로 돌리고, 운영진은 승인하고 체크만 하는 흐름으로 잡아봤어.  
한 번 써보고 솔직하게 뭐가 불편한지 막 말해줘. 그래야 제대로 다듬을 수 있을 것 같아.
